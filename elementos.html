Explicação dos Elementos

Elementos do Projeto

  • Header: A seção de cabeçalho que contém o título e a descrição principal do site.
  • Menu: Um menu suspenso que permite ao usuário navegar pelas diferentes seções do site.
  • Dropdown: Um menu suspenso que exibe as opções de navegação quando clicado.
  • Content: A área principal onde o conteúdo das seções é exibido.
  • Image Container: Um contêiner para a imagem principal do site, centralizada para melhor apresentação.
